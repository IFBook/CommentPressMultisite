== Description ==

Commentpress for Multisite integrates CommentPress in a WordPress Multisite environment and offers various management options.

== Installation ==

1) unzip the .zip file into `/wp-content/plugins/commentpress-mu`
2) Network activate the plugin
3) You are done!
